# Wiki Example Directory

This directory is an example output reference, only for demonstrating the wiki style and structure after generation.

In prompt-only mode, wiki is no longer automatically generated by built-in scripts, but generated by Agent according to the workflow defined in SKILL.md.

## Usage

1. Call generate-wiki skill in the target project
2. Let Agent scan proto and Java source code
3. Agent will generate a similar wiki directory structure

See SKILL.md and README.md in the project root directory for details.

## Notes

- This is an **example output** for reference only — your project's wiki will be different
- Different projects require different documentation depth and structure
- Wiki generation is an **iterative process** — review the output and provide feedback to the Agent for refinement
- The generated wiki should be customized to match your project's specific business domain and team needs
